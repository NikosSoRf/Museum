# Museum
 web приложение для сотрудников музея, в котором перечислены экспонаты коллекции. Для входа в систему сотрудник проходит этап обязательной авторизации, после которой ему доступны CRUD операции и фильтр по категориям над экспонатами музея. Все методы взаимодействия работают по REST API

Документация API доступна в веб - приложении Swagger UI, позволяющим визуализировать определения спецификаций Open API в интерактивном пользовательском интерфейсе по адресу /api-docs/.

Для тестирования и демонстрации работы разработанного REST API был реализован клиентский интерфейс. Вёрстка страниц осуществлена с помощью шаблонизатора EJS. Всего разработано 4 страницы: авторизации, отображения списка экспонатов, добавления и редактирования экспоната.
